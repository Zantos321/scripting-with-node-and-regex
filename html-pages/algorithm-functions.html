functions documented

chunkyMonkey - Type in an array, then type in what size you want the sub arrays to be. Use console.log to see the sub arrays.

mutation - Type in a two element array of strings, if the second string is made up of letters from the first sting, then you will get a true response, if not you will get a false response.

whereDoIBelong - Enter an array of numbers, then a number you want inserted in to array. Hit run to display that number in the index of the sorted array.

falsyBouncer - Type in an array of things. If they are false, it will be filtered out.

sliceSplice - Type in two arrays, then select where you want array 1 to be spliced into array two.

titleCase - Type in a sentence to make the first letter of each word capitalized.

booWho - Type in a boolean true or false to get true, all other input gets false.

findersKeepers - Type in an array and then type 1 to return the first odd number or 2 to return the first even number. If it doesnt it should return an undefined enswer.

stringTruncate - Type in a string the a number value you want that string to be truncated to with ..., if the string is shorter than the number given, i wil just display the string.

stringRepeat - Type in a string then a number, hit run to repeat the string the amount of times specified by you.

endingConfirmed - Type in a string, an then type in another string to see if is the ending of the first string.

largestNumberArrays - Enter one array of numbers, then another array of numbers, then hit run to return a new array with only the largest number in each of the two arrays.

findLongestWordLength - Type a sentence and hit run to find the longest word.

factoralize - Type in a number and hit run to see the product of its factors.

makeStringBackwards - Type in a string and hit run to make it spell out backwards.

celsiusToFahrenheit - Enter the celsius temperature hit run to convert that to fahrenheit.
