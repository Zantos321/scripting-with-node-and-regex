JavaScript Functions: Regex

functions documented

removeWhiteSpace - Type in a string with whitespace at the beginning and end of it and hit run to remove it.

searchAndReplace - Type in three words and hit run to have them returned in reverse order.

mixedGroupingOfCharacters - Type in the name of the president of the 2008 election or his wife's name. Make sure to capitalize the first letter of the first name and last name.

positiveNegativeLookahead - Type in a password that does not start with a number, is at least five characters long, and ends with atleast two numbers.

matchNonWhitespace - Type in a string and hot run to get a count of all scharacters that arent whitespace.

matchNonNumbers - Type in a string and hit run to gat a count of non-nubers in your string.

matchNumbers - Type in a string and hit run to get a count of numbers in your string.

matchAllButLettersAndNumbers - Type in a string and hit run to get the count of all characters that arent letters, numbers, and underscores.

matchAllLettersAndNumbers - Type in a string and hit run to get the character count.

endingStringPatterns - Type in a string and hit run to see if it ends in cat.

beginningStringPatterns - Type in a sentence and then hit run to see if it begins with The.

criminalHunt - We are looking for criminals!, type in a single long string, the Criminals have to be together to be caught. Ex # the criminals in the string need to be like this CCCC!

charctersThatOccurMuli - Type in a string with letters that are back to back to have them show up in the same elements of an array.

notSpecifiedSingleCharacter - Type in a string and hit run to return the chracters that are not vowels or numbers. (this will show spaces too)

singleCharacterMultiPosibilities - Type in a string to pull out the vowels

matchLiteralStrings - Type in a string and hit run, to see if the name Bob is in the string.
